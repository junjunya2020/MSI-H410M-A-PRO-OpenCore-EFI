# 微星 H410M-A PRO OpenCore EFI

## 介绍

- 可用于引导Sur 11.6，其他版本 请自行测试。
- OpenCore 版本:0.6.6
  - 声卡设备地址为`PciRoot(0x0)/Pci(0x1F,0x3)`
  - 使用facetime请自行更换三码
### 正常工作

- [☑️] 板载声卡
- [☑️] 板载网卡
- [☑️] 核显硬解
- [☑️] 睡眠 / 唤醒
- [☑️] USB2.0 / USB 3.0

### 未知问题

- [❓] 声音输入可能有问题，请自行更换layout-id
- [❓]没有购买过蓝牙Wi-Fi模块，类似功能请自己测试
- [❓]facetime未测试
### 未解决问题
- [✖️]启动音频
